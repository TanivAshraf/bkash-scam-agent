About This Project

What is this system?

This is a fully autonomous AI agent designed to act as a digital detective for bKash. Its sole purpose is to automatically patrol the internet, identify websites that are potentially misusing the bKash name to promote betting or gambling scams, and present these findings in a clean, focused, and easy-to-understand web dashboard.

It is a zero-maintenance system; once deployed, it runs entirely on its own without needing a dedicated computer or any manual intervention.

How It Works: A Daily Mission

  1. The Scheduled Wake-Up: Every morning, the system automatically wakes up and begins its mission.
  2. The Hunt (in Two Languages): The system runs two separate "micro-agents" to search Google using both English and Bangla keywords for maximum coverage.
  3. The Investigation: For each URL found, the agent visits the website and reads its complete text content, just like a human analyst would.
  4. The Verdict (The AI Brain): The extracted text is sent to a powerful AI language model (Google's Gemini) which has one critical task: decide if the website is a potential threat.
  5. The Report (Smart Filtering): If the AI determines a site is a potential scam, it's saved to a secure database. If a site is irrelevant or an error, the agent discards it to keep the final report clean and actionable.
  6. The Live Dashboard: The main dashboard provides a live, filtered view of only the potential scam sites the agent has found.

Technology & Design

The system was designed for reliability, focus, and zero maintenance. By using a "Serverless Architecture" with specialized tools for each step (searching, scraping, AI analysis), the agent is robust, scalable, and requires no ongoing management.

Key Technologies: Vercel (Hosting & Scheduling), Supabase (Database), Google Gemini (AI Model), SerpApi & ScraperAPI (Web Data), Python (Agent Logic), and HTML/CSS/JavaScript (Dashboard).
